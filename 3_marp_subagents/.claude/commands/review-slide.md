# review-slide（レビューエージェント）

2_slides/slides.md を検証し、改善提案を行う。

## 手順

1. `2_slides/slides.md` を読み込む
2. `.claude/skills/marp/skill.md` を参照
3. 以下の観点でレビュー
4. 改善提案をレポート

## レビュー観点

### 1. 構文チェック

- [ ] フロントマターが正しい
- [ ] スライド区切り（`---`）が適切
- [ ] Markdown構文エラーがない

### 2. 文字数チェック

- [ ] タイトル: 30文字以内
- [ ] リード文: 2行以内
- [ ] 箇条書き: 1項目25文字以内
- [ ] 箇条書き数: 5項目以内

### 3. 構成チェック

- [ ] タイトルスライドがある
- [ ] アジェンダがある
- [ ] 1スライド1メッセージ
- [ ] まとめスライドがある
- [ ] 3層構造（タイトル・リード文・ボディ）

### 4. デザインチェック

- [ ] 視覚的な階層が明確
- [ ] 余白が適切
- [ ] 画像サイズが適切

## 出力フォーマット

```markdown
## レビュー結果

### 問題点

1. スライド3: タイトルが35文字で制限超過
2. スライド5: 箇条書きが7項目で多すぎ

### 改善提案

1. スライド3: 「〇〇」→「△△」に短縮
2. スライド5: 2スライドに分割

### 総評

（全体的な評価とアドバイス）
```
