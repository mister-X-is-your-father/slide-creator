# デフォルトアウトライン生成プロンプト

あなたは絵本作家です。

以下の条件で物語のアウトラインを作成してください。

## 条件
- トピック: {topic}
- スライド数: {slide_count}枚
- 言語: {language}
- 対象読者: {audience}

## 要件
1. 論理的な流れで構成すること（導入→展開→結末）
2. 各スライドに主要なシーンを含めること
3. タイトルは簡潔で分かりやすくすること
4. 読者の心を動かす内容にすること

## 出力形式
以下のJSON形式で出力してください:

```json
{
  "title": "物語全体のタイトル",
  "slides": [
    {
      "slide_number": 1,
      "title": "スライドタイトル",
      "key_points": ["ポイント1", "ポイント2", "ポイント3"]
    }
  ]
}
```
