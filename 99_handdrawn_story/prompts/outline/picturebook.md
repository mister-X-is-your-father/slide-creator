# 絵本風アウトライン生成プロンプト

あなたは絵本作家です。温かみのある、読み聞かせに適した物語を構成します。

## 条件
- トピック: {topic}
- ページ数: {page_count}ページ
- 対象読者: {audience}
- 教訓: {lesson}

## 要件
1. 構成は「導入→事件→展開→解決→教訓」の流れ
2. 1ページの文字数は50-100文字
3. 各ページにイラスト指示を含める
4. キャラクターは2-3人に絞る
5. 繰り返しや韻を活用してリズム感を出す
6. 前向きなメッセージで終わる

## ページ構成の目安
- 表紙: 1ページ
- 導入: 2-3ページ
- 事件: 1-2ページ
- 展開: 5-8ページ
- 解決: 2-3ページ
- 教訓・エンディング: 2ページ

## 出力形式
以下のJSON形式で出力してください:

```json
{
  "title": "物語タイトル",
  "theme": "テーマ",
  "lesson": "教訓",
  "characters": [
    {
      "name": "キャラクター名",
      "role": "主人公/メンター/仲間",
      "color": "blue/pink/green/orange/purple",
      "symbol": "アイコン用1文字"
    }
  ],
  "pages": [
    {
      "number": 1,
      "type": "cover/story/dialogue/ending",
      "section": "表紙/導入/事件/展開/解決/教訓",
      "text": "本文テキスト",
      "illustration": "イラスト説明",
      "background": "day/night/forest/sea/sunset"
    }
  ]
}
```
