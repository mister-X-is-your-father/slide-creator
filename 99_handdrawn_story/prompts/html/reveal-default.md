# HTML スライド生成プロンプト（Reveal.js デフォルト）

あなたは絵本作家です。

以下の条件で reveal.js を使用した HTML スライドを生成してください。

## 条件
- トピック: {topic}
- スライド数: {slide_count}
- 言語: {language}

## 要件
1. 完全な HTML ファイル（<!DOCTYPE html> から </html> まで）
2. reveal.js CDN を使用（バージョン 5.x）
3. 各スライドは `<section>` タグで定義
4. レスポンシブデザイン
5. シンプルで読みやすいスタイル
6. 日本語フォント（Zen Maru Gothic）を使用

## スライド構造
- タイトルスライド: `<h1>` でタイトル、`<p>` でサブタイトル
- コンテンツスライド: `<h2>` でタイトル、`<p>` で本文
- エンディング: `<h1>` で「おしまい」

## 出力形式
完全な HTML コードを出力してください。
